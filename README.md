# gateway_mido

## Gateway API (8080)

- GET
- 홈 화면
- 로그인 화면
- 회원가입 화면
- MyPage 계정정보 정보, 수정 화면
- MyProject (id에 해당하는 프로젝트들 보여주는 페이지)
- Project (프로젝트 클릭 시 보여주는 페이지, task, milestone, tag 등등 포함)
- Project 생성, 수정 화면
- MyTask(id에 해당하는 task들 보여주는 페이지)
- Task (Task 클릭 시 정보 페이지)
- Task 생성, 수정 화면
- comment 생성, 수정 화면
- tag 생성, 수정화면
- milestone 생성, 수정화면
